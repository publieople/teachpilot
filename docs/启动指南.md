# TeachPilot 启动指南

## 第一步：获取 OpenRouter API Key

1. 访问 https://openrouter.ai/keys
2. 注册/登录账号
3. 创建新的 API Key
4. 复制 Key 保存

**免费额度：** Qwen3.6-Plus:free 有免费额度，适合开发和测试

---

## 第二步：后端启动

### 1. 进入后端目录
```bash
cd backend
```

### 2. 使用 uv 安装依赖
```bash
# 安装 uv (如果未安装)
pip install uv

# 创建虚拟环境并安装依赖
uv venv
.\venv\Scripts\Activate.ps1  # PowerShell
# 或
venv\Scripts\activate  # CMD

# 安装项目依赖
uv pip install -e .
```

**或者更简单的方式（推荐）：**
```bash
# 一条命令完成虚拟环境创建 + 依赖安装
uv sync
```

### 4. 配置环境变量
```bash
# 复制示例文件
copy .env.example .env

# 编辑 .env 文件，填入你的 API Key
# OPENROUTER_API_KEY=your_key_here
```

### 5. 启动服务
```bash
python main.py
```

**成功标志：**
```
🚀 TeachPilot 启动中...
📖 API 文档：http://localhost:8000/docs
INFO:     Uvicorn running on http://0.0.0.0:8000
```

访问 http://localhost:8000/docs 查看 API 文档

---

## 第三步：前端启动

### 1. 进入前端目录
```bash
cd frontend
```

### 2. 安装依赖
```bash
npm install
```

### 3. 启动开发服务器
```bash
npm run dev
```

**成功标志：**
```
  VITE v5.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

访问 http://localhost:5173

---

## 第四步：测试对话功能

1. 打开浏览器访问 http://localhost:5173
2. 在对话框输入："我想设计一节关于 Python 基础语法 的课程"
3. 等待 AI 回复

**如果看到错误 "OpenRouter API Key 未配置"：**
- 检查 `backend/.env` 文件是否存在
- 检查 `OPENROUTER_API_KEY` 是否正确填写
- 重启后端服务

## 常见问题

### Q: uv pip install 失败
A: 尝试使用国内镜像：
```bash
uv pip install -e . -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q: npm install 失败
A: 尝试使用淘宝镜像：
```bash
npm config set registry https://registry.npmmirror.com
npm install
```

### Q: 端口被占用
A: 修改 `backend/config.py` 中的 `PORT` 或 `frontend/vite.config.ts` 中的 `port`

### Q: CORS 错误
A: 开发环境已配置允许所有来源，生产环境需要修改 `backend/main.py` 中的 CORS 配置
