# TeachPilot 技术栈

**更新时间：** 2026-04-04  
**赛题：** 大学生服务外包创新创业大赛 A04 - 多模态 AI 互动式教学智能体  
**企业：** 锐捷网络

---

## 🧠 大模型 (LLM)

| 项目 | 选型 |
|------|------|
| **主模型** | Qwen3.6-Plus (通义千问) |
| **API 提供商** | OpenRouter (免费额度) |
| **模型 ID** | `qwen/qwen3.6-plus:free` |
| **备用方案** | 阿里云百炼 API (付费，约 0.004 元/K tokens) |
| **多模态能力** | 文 + 图/视频 → 文，视觉推理能力飞跃式进步 |

**OpenRouter 配置：**
```bash
OPENROUTER_API_KEY=your_key
OPENROUTER_BASE_URL=https://openrouter.ai/api/v1
MODEL_ID=qwen/qwen3.6-plus:free
```

## 🔤 Embedding 模型

| 项目 | 选型 |
|------|------|
| **模型** | BGE-M3 (北京智源 AI 研究院) |
| **HuggingFace** | `BAAI/bge-m3` |
| **最大上下文** | 8192 tokens |
| **支持语言** | 多语言（中文优化） |
| **部署方式** | 本地部署 (sentence-transformers) |
| **备用方案** | Qwen3-Embedding-0.6B |

**安装：**
```bash
pip install sentence-transformers
```

**使用示例：**
```python
from sentence_transformers import SentenceTransformer
model = SentenceTransformer('BAAI/bge-m3')
embeddings = model.encode(["你的文本"])
```

## 📚 向量数据库

| 项目 | 选型 |
|------|------|
| **数据库** | ChromaDB |
| **部署方式** | 嵌入式 (PersistentClient) |
| **适用场景** | 比赛/原型开发 |
| **备用方案** | Milvus (生产环境) |

**优势：**
- 轻量级，开箱即用
- Python 原生，文档完善
- 支持持久化存储
- 适合中小规模数据（比赛场景足够）

**安装：**
```bash
pip install chromadb
```

## 🌐 前端

| 项目 | 选型 |
|------|------|
| **框架** | React 18 + TypeScript |

- ⚡ 秒级启动，热更新快

- 生态成熟，组件丰富

**安装：**
```bash
cd frontend
npm install
```

## 📎 多模态解析

| 格式 | 库 | 说明 |
|------|-----|------|
| **PDF** | PyMuPDF (`fitz`) | 速度快，支持 OCR，跨平台 |
| **Word** | python-docx | 成熟稳定 |
| **PPT** | python-pptx | 读写支持，跨平台 |
| **视频** | OpenCV + ffmpeg | 关键帧提取 + 摘要 |
| **图片** | Pillow | 基础处理 |

**安装：**
```bash
pip install PyMuPDF python-docx python-pptx opencv-python pillow
```

## 📊 PPT 生成

| 项目 | 选型 |
|------|------|

## 📦 项目结构

```
teachpilot/
├── backend/              # Python 后端
│   ├── main.py          # FastAPI 入口
│   ├── requirements.txt # Python 依赖
│   ├── modules/         # 功能模块
│   │   ├── chat/        # 对话管理
│   │   ├── rag/         # RAG 检索
│   │   ├── multimodal/  # 多模态解析
│   │   └── generator/   # 课件生成
│   └── knowledge-base/  # 本地知识库
├── frontend/            # React 前端
│   ├── src/
│   │   ├── components/  # 组件
│   │   ├── pages/       # 页面
│   │   ├── stores/      # 状态管理
│   │   └── services/    # API 调用
│   └── package.json
├── docs/                # 项目文档
└── scripts/             # 工具脚本
```

## 🚀 快速开始

### 后端
```bash
cd backend
pip install -r requirements.txt
python main.py
# 访问 http://localhost:8000/docs 查看 API 文档
```

### 前端
```bash
cd frontend
npm install
npm run dev
# 访问 http://localhost:5173
```

## 📋 技术选型理由总结

| 组件 | 选型 | 关键理由 |
|------|------|----------|
| LLM | Qwen3.6-Plus | 多模态能力强，OpenRouter 免费 |
| Embedding | BGE-M3 | 中文 SOTA，8192 上下文，本地免费 |
| Vector DB | ChromaDB | 轻量、易用、适合比赛 |
| Backend | FastAPI | 异步、自动 API 文档 |
| Frontend | Vite + React | 开发效率高，生态成熟 |
| PDF | PyMuPDF | 速度快，支持 OCR |
| PPT | python-pptx | 跨平台，够用 |
| Voice | Web Speech + Edge TTS | 全免费方案 |
